## ðŸ§¹ Methods

### **Data Source**
The dataset used (`IMDb Horror: Chilling Movie Dataset') contains movie titles, release years, genres, IMDb ratings, 
votes, runtime, directors, and gross revenue. The dataset reflects modern IMDb user ratings rather than 
1980s audience reactions.

### **Data Cleaning**
- Column names were normalized by converting all names to lowercase and replacing spaces with underscores.
- Numeric columns (`rating`, `votes`, `movie_year`) were converted using `pd.to_numeric` with coercion.
- Rows with invalid or missing release year values were removed for consistency.

### **Filtering and Subsetting**
- A horror-specific subset (`df_h`) was created by selecting rows where the `genre` field contained 
  the substring `"horror"`.
- A `decade` column was generated by flooring each release year to the nearest decade.
- Only films with `decade == 1980` were included for 1980s analysis.

### **Ranking Method**
- Films were ranked primarily by IMDb rating (descending).
- Vote count was used as a secondary sort to favor films with broader audience engagement.
- The Top 10 list reflects the highest-rated horror films of the decade according to modern user ratings.
